# PyfagorAI — Telegram бот для обучения математике с использованием Gemini AI

Бот-репетитор, который помогает решать математические задачи с помощью текстового или фото ввода. Использует Google Gemini AI для генерации ответов, поддерживает контекст диалога и не имеет встроенную систему подписок (Пока что).

## ✨ Возможности

- 📝 Решение текстовых математических задач
- 📷 Распознавание и решение задач с фотографий
- 💬 Поддержка контекста диалога
- ⚙️ Переключение режимов работы (tutor и другие)
- 🧹 Сброс истории диалога
- 🏠 Удобное меню с кнопками

## 🛠 Технологии

- Python 3.10+
- Aiogram 3.x
- Google Gemini AI
- Aiohttp (для healthcheck)
- Telegram Bot API

🎮 Команды бота

· /start — Начало работы, приветствие

🧠 Как это работает

1. Пользователь отправляет текст задачи или фотографию
2. Бот сохраняет контекст диалога
3. Запрос отправляется в Google Gemini AI с учетом выбранного режима
4. Ответ приходит в формате HTML с пояснениями
5. История диалога сохраняется для поддержания контекста

💳 Подписка (PRO режим) (не реализовано)

· Интеграция с Telegram Stars
· Доступ к расширенным возможностям:
  · Все предметы (включая университетский уровень)
  · Анализ ошибок
  · Пошаговое объяснение
  · Альтернативные методы решения

⚠️ Обработка ошибок

Бот обрабатывает ошибки Google API:

· 400 — повторите позже
· 429 — перегрузка бота
· 503 — проблемы с сервером

🧹 Сброс контекста

Команда /reset очищает историю диалога для текущего пользователя.

🌐 Healthcheck

На порту PORT запущен простой HTTP сервер для проверки состояния бота (URL: /).
